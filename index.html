Publication

DESIGN OF USER INTERFACE DESIGN OF ENDLESS RUNNER GAME BASIC MATHEMATICS BASED ON ANDROID USING SYSTEM USABILITY SCALE METHOD

2024

Jurnal Adat

  • This research designs and evaluates the user interface of an educational mathematics game based on Android in the endless runner genre using the System Usability Scale (SUS) method to measure usability.
  • The game achieved an average SUS score of 88.92 in the "Excellent" category, with strengths in its simple interface layout, intuitive navigation, and optimal button responsiveness.
  • The research provides recommendations to improve visuals, add dynamic difficulty levels, and serve as a reference for developers in designing user-friendly educational games.

Transfer Learning and Fine-tuning Effect Analysis on Classification of Cat Breeds using a Convolutional Neural Network

2023

6th International Conference on Information and Communications Technology (ICOIACT) 2023

  • This research aimed to analyze the effect of transfer learning and fine-tuning on CNN's performance in classifying 12 cat breeds from the Oxford-IIIT Pet dataset, totaling 2400 images.
  • Implementing CNN with transfer learning and fine-tuning achieved a 92.5% accuracy rate on test data, highlighting the significance of these methods in improving model performance.
  • Transfer learning and fine-tuning are crucial for enhancing CNN's efficiency in cat breed classification. Future research could focus on improving the model, adding more cat breed classes, and exploring other architectures and optimizations for better outcomes.

Comparative Analysis of Software Development Lifecycle Methods in Software Development: A Systematic Literature Review

2023

International Journal of Advances in Data and Information Systems (IJADIS)

  • This research compares various Software Development Lifecycle (SDLC) models such as Waterfall, Agile, and Scrum, focusing on flexibility, speed of development, adaptation to changing requirements, and project risk.
  • The results show that each method has advantages and disadvantages: Agile and Scrum emphasize flexibility and teamwork, while Waterfall offers greater structure and clarity of plans.
  • The research aimed to determine the best process method for software development, with Agile found to be the most widely used method due to its efficiency and effectiveness in software development.

Sentiment Analysis for IMDb Movie Review Using Support Vector Machine (SVM) Method

2023

Inform: Jurnal Ilmiah Bidang Teknologi Informasi dan Komunikasi

  • This research uses supervised machine learning methods, specifically Support Vector Machines (SVM), for sentiment analysis on Twitter and IMDb datasets, with a focus on movie reviews.
  • By preprocessing and classifying 50,000 data (40,000 for training and 10,000 for testing, divided into 25,000 positive and 25,000 negative), this research evaluates the model using accuracy, precision, recall, and F1 score.
  • Using Bags of Words (BoW), the model achieved 88.59% accuracy, while parameter optimization with grid-search and the use of Term Frequency–inverse Document Frequency (TF-IDF) increased the accuracy to 91.27%.

Comparative Analysis of Optimizers on NASNetMobile Convolutional Neural Network Architecture for Cat Breed Classification

2022

Jurnal Teknologi

  • This research explores the use of deep learning, specifically Convolutional Neural Networks (CNN) with the NASNetMobile architecture, to classify cat breeds.
  • The goal is to help cat owners understand their cat's breed, care, activities, and personality.
  • This research dataset consists of 840 images divided into 4 classes, with 588 training data, 168 test data, and 84 validation data.
  • The results show that using the RMSprop optimizer with a learning rate of 0.0001 provides the highest accuracy of 89.88%, while SGD provides the lowest accuracy of 78.57%. This shows the importance of choosing the right architecture and optimizer in improving the performance of cat breed classification models.
